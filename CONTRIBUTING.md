# Contributing

Thanks for helping improve 提瓦特微信.

Before opening a pull request:

1. Run `flutter analyze`.
2. Build at least once with `flutter build apk --release` when touching Android or runtime logic.
3. Do not commit APK files, local chat data, cache folders, or API keys.
4. Keep character replies short and natural. Avoid AI assistant phrasing.
5. For group chat changes, verify that not every character replies every round.

This is an unofficial fan project. Please keep third-party copyright and attribution boundaries clear.
