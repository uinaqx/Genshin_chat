# Security

## API Key

This app expects users to provide their own OpenAI-compatible LLM API Key.

- Do not commit real API keys.
- Do not paste API keys into GitHub Issues.
- Do not upload screenshots that reveal API keys.
- Do not add logs that print request headers or complete request bodies.

## Local Data

Chat history and settings are local app data. They should not be committed to this repository.

If you discover a security issue, create a private report or contact the maintainer directly instead of opening a public issue with sensitive details.
